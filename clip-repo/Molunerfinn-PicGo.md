## Backup commandline

### [[windows]]

```
%APPDATA%\picgo\data.json
```

### [[custom/linux|linux]]

```
$XDG_CONFIG_HOME/picgo/data.json` or `~/.config/picgo/data.json
```

### [[macos]]

```
~/Library/Application\ Support/picgo/data.json
```

via: https://picgo.github.io/PicGo-Doc/en/guide/config.html#basic-operations
